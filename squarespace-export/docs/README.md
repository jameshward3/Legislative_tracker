# City of Orange Legislative Tracker

Static GitHub Pages build for the City of Orange Legislative Tracker.

Embed URL:

`https://jameshward3.github.io/Legislative_tracker/`

Primary files:

- `index.html`
- `styles.css`
- `app.js`
- `seed-data.js`
- `imported-history-data.js`
- `budget-dashboard-data.js`
- `assets/`

The app runs fully in the browser and stores local admin edits in browser `localStorage`.
